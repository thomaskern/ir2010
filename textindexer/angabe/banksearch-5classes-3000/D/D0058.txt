# Java Bytes features items of interest for novices and experts, as
well as the rank beginner who just wants an easy way to get a few applets on
their sites.
http://javaboutique.internet.com/javabytes/
:: CorbaTrace
CorbaTrace is a helpful tool for tracing communications beetween Corba 
objects. Once CorbaTrace is installed, remote calls are intercepted and all 
information is stored in XML log files. After that, you will be able to 
apply filters and produce XML files to see the communications beetween 
distant objects on a UML sequence diagram.
http://jars.com/classes/jresout.cgi?resource=10197
:: Intelligent Meta Tag Generator
XDE's meta tag generator is the orginal Java 1.3 application for analyzing 
lists of competitor web sites and generating HTML META tags suitable for 
inclusion in your HTML document. These tags allow better indexing by 
robot-driven search engines, such as AltaVista, Google, InfoSeek and searchBC.
http://jars.com/classes/jresout.cgi?resource=10194
:: SnafuTicker
SnafuTicker is a light-weight horizontically text scrolling/fading applet. 
Loads fast with only 7 db as .jar-file (12 kb uncompressed) & is 
configurable directly through HTML.
http://jars.com/classes/jresout.cgi?resource=10195
:: SMTP Firewall
Provided by: Charles Anderson - XDE's Extensible Java JDK 1.3 SMTP firewall 
is a smtp relay mechanism which listens on port 25 and forwards emails 
after filtering to an alternate smtp server on a different ip or port.
http://jars.com/classes/jresout.cgi?resource=10191
:: Voice Conference API
SmileTiger Voice API" is a voice conference API based on JMF and Java 
Servlet technologies. This API handles dynamic group voice conferences. It 
is built on standards like Java/J2EE, JMF, RTP/RTCP, GSM, HTTP and XML.
/http://jars.com/classes/jresout.cgi?resource=10179
